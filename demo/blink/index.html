Blink

Description

The blink pattern signals importance or urgency by blinking the content of an element.

Demo

There are three ways to invoke the blink pattern.

  1. The classical blink element: 
    <blink>The classical blink element:</blink>
  2. A modern web component variant: 
    <x-blink>A modern web component variant:</x-blink>
  3. And Patterns-style: 
    <span class="pat-blink">And Patterns-style:</span>

For all variants, the classical Mozilla timing has been chosen.
