pat-display-time demo

Example of default

Code

        
            <time class="pat-display-time"
                  datetime="2015-01-20T08:00Z">
                20 January 2015, 08:00
            </time>

Output

20 January 2015, 08:00

Example setting the output format explicitly

Code

        
            <time class="pat-display-time"
                  datetime="2015-01-20T08:00Z"
                  data-pat-display-time="output-format: MMMM dddd Do YYYY, h:mm:ss a">
                20 January 2015, 08:00
            </time>

Output

20 January 2015, 08:00

Output formatted as local

Code

        
            <time class="pat-display-time"
                  datetime="2015-01-20T08:00Z"
                  data-pat-display-time="output-format: L">
                20 January 2015, 08:00
            </time>

Output

20 January 2015, 08:00

Output formatted as 'from now'

Code

        
            <time class="pat-display-time"
                  datetime="2015-01-20T08:00Z"
                  data-pat-display-time="from-now: true">
                20 January 2015, 08:00
            </time>

Output

20 January 2015, 08:00

Output formatted as 'from now' with no suffix

Code

        
            <time class="pat-display-time"
                  datetime="2015-01-20T08:00Z"
                  data-pat-display-time="from-now: true; no-suffix: true">
                20 January 2015, 08:00
            </time>

Output

20 January 2015, 08:00

Output formatted as 'from now' with no suffix in german

Code

        
            <time class="pat-display-time"
                  datetime="2015-01-20T08:00Z"
                  data-pat-display-time="from-now: true; no-suffix: true; locale: de">
                20 January 2015, 08:00
            </time>

Output

20 January 2015, 08:00
